# Crypto Market Dashboard 📊

A real-time cryptocurrency dashboard consuming a public API to display live prices, market cap and 24h changes for the top 20 coins.

🔗 **Live Demo:** [https://ale13211.github.io/crypto-dashboard/](https://ale13211.github.io/crypto-dashboard/)

---

## ✨ Features

- Real-time prices for top 20 cryptocurrencies
- Live market cap, 24h volume, gainers & losers stats
- Price flash animation on update (green/red)
- Search by name or symbol
- Skeleton loading screen
- Auto-refresh every 30 seconds
- Fully responsive design
