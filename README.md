# Panel de control del mercado de criptomonedas

Un panel de control de criptomonedas adaptable que obtiene datos de mercado en tiempo real mediante una API pública.

## Descripción

Este proyecto demuestra habilidades modernas de desarrollo front-end, incluyendo el consumo de API, actualizaciones dinámicas de la interfaz de usuario y diseño responsivo. Destaca las herramientas y tecnologías más valoradas por los reclutadores para desarrolladores front-end junior/intermedios.

## Herramientas y tecnologías

- **HTML5 y CSS3**: Para estructurar y dar estilo a una interfaz de usuario limpia y adaptable.

- **JavaScript (ES6+)**: Lenguaje principal para la funcionalidad dinámica y el consumo de API.

- **API CoinGecko**: Proporciona datos de criptomonedas en tiempo real para una aplicación basada en datos del mundo real.

- **Principios de diseño responsivo**: Garantiza la usabilidad en diferentes dispositivos, una habilidad fundamental para el desarrollo web moderno.
